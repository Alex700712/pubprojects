#ИНСТРУКЦИЯ ПОЛЬЗОВАТЕЛЯ

Приложение для управления ключами организаций с консольным меню 

Необходимые элементы: 
PostgreSQL
Java (Приложение создано в Intelij IDEA)

ИНСТРУКЦИЯ:
Шаг 1: Создайте базу данных при помощи SQL запроса из файла SQL testdb PostgreSQL. 
Шаг 2: Скачайте и запустите проект test 1.
Шаг 3: Измените данные в свойствах username и password на данные своего подключения БД (в файле hibernate.cfg.xml).
Шаг 4: Запустите приложение.

Функционал:
Операции добавления, удаления, редактирования и вывод значений для БД с организациями; 
Операции добавления, удаления, редактирования и вывод значений для БД с ключами;
Вывод отсортированного списка ключей с фильтром по организации (её идентификатору);
Вывод найденного значения из таблицы ключей по конкретному ключу (UUID);
